#### ⚠️ 迁移进度

**需检查**：是否各模块都已通过 `AppInit.register()` 注册？

当前 index.html 脚本加载：
```html
<script src="js/core/config.js"></script>
<script src="js/core/leletv-global.js"></script>
<script src="js/core/storage-service.js"></script>
<script src="js/core/listener-tracker.js"></script>
<script src="js/core/timing.js"></script>
<script src="js/api/api-config.js"></script>
...（还有 15+ 个脚本）
```

**目前状态**：
- ✅ 脚本顺序已优化（依赖关系清晰）
- ✅ 使用了 `defer` 属性
- ⚠️ 但还没有将**所有初始化**迁移到 `AppInit` 中
- ⚠️ 各模块仍然各自 listen 到 `DOMContentLoaded`，而不是注册到 `AppInit`

**建议修复步骤**：
```javascript
// 旧方式（当前）
document.addEventListener('DOMContentLoaded', function() {
    initSearchModule();
    initTmdbModule();
});

// 新方式（AppInit）
AppInit.register('search', AppInit.PHASES.CONFIG, function() {
    initSearchModule();
});

AppInit.register('tmdb', AppInit.PHASES.CONFIG, function() {
    initTmdbModule();
});

// 统一在 index.html 中调用
document.addEventListener('DOMContentLoaded', function() {
    AppInit.run();
});
```
