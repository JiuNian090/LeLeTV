## 🛠️ 本地开发指南

如需在本地进行开发和测试，请按照以下步骤操作：

1. 克隆本仓库
   ```bash
   git clone <your-repo-url>
   cd JNTV-t
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 配置环境变量
   ```bash
   cp .env.example .env
   # 根据需要修改 .env 文件中的配置，特别是设置 PASSWORD 变量
   ```

4. 启动开发服务器
   ```bash
   npm run dev
   ```

5. 访问应用
   ```
   打开浏览器访问 http://localhost:8080
   ```
